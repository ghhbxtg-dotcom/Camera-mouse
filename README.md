# Camera-mouse
Управление мышью жестами с помощью веб камеры. Перемещайте курсор рукой и выполняйте щелчки мышью простыми движениями пальцев. Создано на основе Python, OpenCV и MediaPipe

## Как пользоваться

1. Скопируйте репозиторий или загрузите ZIP-файл
2. Извлеките файлы в удобную папку(главное - извлеките все файлы)
3. Запустите run.bat (Windows), нажмите 2 раза enter и подождите(для установки зависимостей)или используйте Python.
4. Управляйте курсором рукой:
5. Переместите указательный палец → переместите курсор
Сожмите указательный + большой палец - ЛКМ
Сожмите средний + большой палец - ПКМ
4. Нажмите клавишу ESC для выхода
